to:
```

```

Then update the useEffect body (around lines 24-27). Change:
```
fetchRecentBlogPosts().then(posts => {
  setRecentPosts(posts.slice(0, 3));
});
```
to:
```
getAllPosts().then(posts => {
  const sorted = [...posts].sort(

  setRecentPosts(sorted.slice(0, 3));
});
```
This replicates the exact behavior — `fetchRecentBlogPosts` internally fetched all posts, sorted by publishedDate descending, and returned the top 3. `getAllPosts` from blogUtils uses the same data source (API with `/blog-posts.json` fallback).
