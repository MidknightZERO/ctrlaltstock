Create a new component HomepageBackground.tsx that renders a full-viewport, fixed, pointer-events-none layer behind the main content. Use one of:
- Matrix-style: Very subtle falling characters (e.g. 0/1 or small symbols) with CSS animation (opacity ~0.03–0.06, brand color #9ed04b or gray), low density so it does not distract.
